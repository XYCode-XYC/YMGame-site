私たちは現在それがどうなるかを開発していますか？

—————————————————————————————————————————————————— ————————

  • done 予備レビュー

    予備審査はすべてのオンラインゲームに必要なステップです。予備審査に合格した後、NetEaseはオンラインゲームの開発とテストのための方法を開発します。

  • replay 開発テストサービス

    私たちのプログラマー（XYCode）は、YM Gameでさまざまなコードを記述し、それらを開発マシンにデプロイします。

    次に、テスターはMOD-PCまたは私の世界のベータ版でYMゲームベータ版にログインします。

    BUGをテストするとき、ティックリストによるテスターはBUG FIXミッションを送信し、テストを再度修復するプログラマーになります。BUGがない場合、このBUGFIXタスクのBUGは完了状態としてマークされます。

  • power_settings_new 公式サーバーを開発する

    主なコンテンツは、公式サービステストサービスを開発し、コンテンツを公式サービスに移行することです。公式サーバーのサーバーは一般的に次のとおりです。

    構成項目 構成内容 備考
    パフォーマンスタイプ 低頻度
    メモリタイプ 128G
    ハードディスクタイプ 800G SSD
    アプリケーションの数 1セット 後の状況に応じて適切に増加します

  • power_settings_new 正式にローンチ

    YMゲームはまだオンラインではありません。オンラインになった後は、誰でも自由にオンラインゲームに参加してプレイできます。
