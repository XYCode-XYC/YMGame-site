我们目前 开发的怎么样了？

————————————————————————————————————————————————————————————

  • done 初审

    初审是每个网络游戏的 必须步骤， 通过初审之后，网易会为我们方法开发机，用于网络游戏的开发与测试。

  • replay 开发测试服

    我们的程序员（XYCode）将在 YM Game 内编写各种代码，并将他们部署到开发机。

    接下来，测试员将会通过 MOD-PC 或 我的世界测试版 登录到 YM Game-测试版。

    当测试出 BUG 的时候，测试员通过 滴答清单 发送 BUG FIX 任务，待程序员修复后再次测试，当没有 BUG 时，关于此 BUG 的 BUG FIX 任务将会被标记为 完成 状态。

  • power_settings_new 开发正式服

    开发正式服的主要内容就是将 测试服 的内容迁移到 正式服。 正式服的服务器一般为如下：

    配置项 配置内容 备注
    性能类型 低频
    内存类型 128G
    硬盘类型 800G SSD
    申请数量 1 台 数量会根据后期情况适量增加

  • power_settings_new 正式上线

    目前 YM Game 暂未上线，上线之后大家即可自由进入网络游戏中游玩。
